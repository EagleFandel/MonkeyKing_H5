# 四大场景流程线框（文本版）

## 全局入口流
`首页` -> 选择场景 -> `需求/画像页面` -> 智能体补全 -> `匹配结果` -> `详情/动作` -> `支付或申请` -> `消息与个人中心沉淀`

## 场景一：创业者找开发者
1. 进入 `find-project`
2. 填写项目表单
3. 智能体提取与追问
4. 生成报告
5. 匹配候选列表
6. 查看候选详情
7. 选择候选创建订单
8. 跳转支付并完成

## 场景二：家长找家教
1. 进入 `find-tutor`
2. 填写需求表单
3. 智能体补全
4. 匹配家教候选
5. 选择候选创建联系方式订单
6. 跳转支付并完成

## 场景三：网球陪练
1. 进入 `find-sparring`
2. 填写陪练需求
3. 智能体补全
4. 匹配陪练候选
5. 选择候选创建预约订单
6. 跳转支付并完成

## 场景四：大学生找兼职
1. 进入 `discover-jobs`
2. 填写技能与偏好
3. 智能体补全
4. 生成画像
5. 推荐兼职列表
6. 一键申请并提示结果

## 通用保障流
- 实名认证流：`profile/auth` -> 提交认证 -> 成功后解锁关键动作
- 投诉流：`complaints` -> 填写证据 -> 返回投诉编号
- 消息流：`messages` -> 刷新 -> 标记已读
- 返利流：`profile` -> 查看返利 -> 提现
