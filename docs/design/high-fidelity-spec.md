# 高保真页面规范（文档版）

## 页面状态统一规范
每个关键页面至少覆盖以下状态：
1. 默认态：展示主信息与主操作。
2. 加载态：请求中（按钮 loading 或骨架）。
3. 空态：无候选/无消息/无记录。
4. 错误态：接口失败，toast + 可重试按钮。
5. 成功态：提交/支付/申请成功反馈。

## 1. 首页（`/`）
- 顶部：Logo + 消息。
- 主区：欢迎语、一句核心价值、主 CTA。
- 中区：四场景快捷卡片（图标+标题+说明）。
- 下区：三大优势（智能匹配、实名认证、返利）。

## 2. 场景页（四类）

### 2.1 创业者找开发者（`/find-project`）
- 表单区：项目类型、技术栈、预算、周期、性格偏好、描述。
- 智能体区：自然语言输入，回填预算/描述。
- 报告区：生成报告 + 下载链接。
- 候选区：匹配候选卡片列表。

### 2.2 家长找家教（`/find-tutor`）
- 表单区：年级、科目、时段、方式、地点、预算、个性需求。
- 智能体区：提取时段、预算、补问。
- 候选区：家教卡片列表，点击触发联系方式付费。

### 2.3 网球陪练（`/find-sparring`）
- 表单区：运动类型、时间地点、水平、预算、提升方向。
- 智能体区：提取时间/预算/目标。
- 候选区：陪练卡片列表，点击触发预定支付。

### 2.4 大学生找兼职（`/discover-jobs`）
- 表单区：技能、熟练度、时间、薪资、兴趣、介绍。
- 智能体区：提取薪资与介绍。
- 画像区：结构化画像展示。
- 推荐区：兼职卡片列表，点击触发申请。

## 3. 通用页

### 3.1 消息中心（`/messages`）
- 顶部操作：刷新、全部已读。
- 列表：标题、摘要、时间、未读红点。

### 3.2 个人中心（`/profile`）
- 身份卡：角色、实名状态。
- 返利卡：累计/待到账/可提现。
- 快捷入口：消息、投诉、支付。

### 3.3 认证页（`/auth`）
- 角色选择 + 姓名 + 证件 + 自拍 token。
- 成功后返回个人中心并更新状态。

### 3.4 投诉页（`/complaints`）
- 场景、对象ID、原因、证据。
- 提交后显示投诉编号。

### 3.5 支付页（`/payment`）
- 当前订单信息。
- 支付面板：渠道选择 + 确认/取消。
- 历史订单列表。

### 3.6 候选详情页（`/candidate/:id`）
- 基础信息、标签、亮点列表。
