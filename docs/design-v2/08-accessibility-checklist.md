# 悟空哆哆 H5 Design V2 · 可访问性清单

## 1. 颜色与对比
- 正文对比度 >= 4.5:1。
- 大字号文本对比度 >= 3:1。
- 仅用颜色表达状态时，必须补充图标或文案。

## 2. 触控与可操作性
- 所有可点击区域不小于 `44x44`。
- 底部主操作按钮需可单手触达。
- 连续表单项之间间距 >= 8px，防误触。

## 3. 键盘与焦点
- 可聚焦元素必须有可见焦点样式。
- 弹层打开后焦点应落在首个可操作元素。
- Esc/返回应可关闭弹层并恢复焦点。

## 4. 语义与结构
- 页面主标题使用 `h1`，分区标题层级递进。
- 图标按钮提供 `aria-label`。
- 状态区（错误、成功）使用语义提示容器。

## 5. 动效与可达性
- 遵循 `prefers-reduced-motion`，降低动画幅度。
- 避免连续闪烁、频繁视差滚动。

## 6. 状态完整性
- 每个关键页面有 5 态，且状态切换不丢上下文。
- 错误态必须给出恢复操作（重试/返回/联系客服）。
