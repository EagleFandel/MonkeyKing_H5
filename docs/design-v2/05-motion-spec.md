# 悟空哆哆 H5 Design V2 · 动效规范

## 1. 动效目标
- 提升操作可感知性，强化“已响应、处理中、已完成”的心理模型。
- 通过节奏感塑造品牌高级感，避免模板化“瞬切”。

## 2. 时长与节奏
- 元件反馈：`120ms ~ 180ms`
- 区块入场：`320ms ~ 480ms`
- 页面切换：`400ms ~ 550ms`
- 成功态仪式反馈：`600ms ~ 900ms`

## 3. 动效原则
1. 同层级同动效，不混用冲突曲线。
2. 优先位移 + 透明度，谨慎大缩放。
3. 不做无意义循环闪烁。
4. 支持 `prefers-reduced-motion`，降级为淡入。

## 4. 推荐预设

### 4.1 `lift`
- 适用：卡片 hover/press。
- 表现：`translateY(-2px)` + 阴影增强。

### 4.2 `fade-slide`
- 适用：区块入场。
- 表现：`opacity 0->1` + `translateY(8px->0)`。

### 4.3 `scale-pop`
- 适用：标签、pill、数字跳变。
- 表现：`scale(0.96->1)` + 透明度淡入。

### 4.4 `success-burst`
- 适用：支付成功、提交成功。
- 表现：背景光圈扩散 + icon 轻弹。

## 5. 页面转场约定
- 由路由 `meta.motionProfile` 控制。
- `gentle`：普通列表页。
- `immersive`：首页、支付结果页。
- `snappy`：工具页（消息、投诉）。

## 6. 关键交互动效表
| 交互 | 动效 | 时长 |
|---|---|---|
| 按钮点击 | 轻缩放 + 阴影变化 | 120ms |
| Tab 切换 | 色块滑块过渡 | 180ms |
| 列表加载完成 | 卡片错峰淡入 | 320ms |
| 支付成功 | success-burst | 680ms |
| 错误提醒 | 左右轻抖（1次） | 220ms |
