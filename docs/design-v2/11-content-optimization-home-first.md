# 悟空哆哆 H5 Design V2 · 首页内容优化（优先）

## 1. 改造目标
- 从“功能堆叠”升级为“价值叙事 + 快速分流 + 明确行动”。
- 在首屏 5 秒内回答：平台价值、适用人群、下一步动作。

## 2. 首页信息层重排
1. Hero：结果承诺（30 秒说清需求、3 分钟拿到候选建议）。
2. 身份分流条：创业者/家长/陪练需求方/大学生。
3. 智能助手入口：作为主引导之一，不再是弱入口。
4. 四场景卡片：每卡增加适合人群、预计耗时、首个动作。
5. 信任区：实名认证、支付安全、投诉机制。
6. 社会证明：实时数据 + 最近成功案例。

## 3. 文案策略
- 规则：结果导向 + 动词开头 + 低理解成本。
- CTA 统一：`开始匹配`、`马上描述需求`、`查看候选人`。
- 避免：纯功能说明、无动作指令、过长句。

## 4. 四场景页优化建议（P1）
- 表单首屏仅保留必填，选填折叠。
- 标题下固定补充“完成本页你将得到什么”。
- 列表卡片补充“推荐理由一句话”。

## 5. 支付与消息优化建议（P1）
- 支付页增加费用构成、失败恢复路径。
- 消息页增加类型筛选（系统/互动/返利）。
