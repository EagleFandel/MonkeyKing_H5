# 悟空哆哆 H5 Design V2 · 品牌视觉语言

## 1. 品牌风格定位
- 风格关键词：`年轻`、`可信`、`向上`、`有陪伴感`。
- 视觉方向：品牌插画感（代码绘制 + 轻量 SVG），避免图库模板风。
- 品牌延续：保留橙蓝双主轴，升级为“主色 + 功能色 + 氛围色 + 中性色”四层系统。

## 2. 色彩语言

### 2.1 主色（Orange Axis）
- `brand-orange-700`：#E65F00（按压/强调）
- `brand-orange-600`：#FF7A00（主 CTA）
- `brand-orange-500`：#FF922E（悬浮）
- `brand-orange-100`：#FFF2E6（浅底）

### 2.2 辅色（Blue Axis）
- `brand-blue-700`：#2F6DB6（深辅助）
- `brand-blue-600`：#5BA8FF（次 CTA / 链接）
- `brand-blue-500`：#80C0FF（轻交互）
- `brand-blue-100`：#EAF4FF（浅底）

### 2.3 功能色
- Success：#16A34A
- Warning：#F59E0B
- Error：#EF4444
- Info：#3B82F6

### 2.4 中性色
- `neutral-900`：#111827（主标题）
- `neutral-700`：#374151（正文）
- `neutral-500`：#6B7280（次文本）
- `neutral-300`：#D1D5DB（描边）
- `neutral-100`：#F3F4F6（分割）
- `neutral-0`：#FFFFFF（卡片）

## 3. 光感与材质
- 顶部 Hero 使用“橙蓝叠加径向渐变 + 轻噪点”构建品牌氛围。
- 卡片采用“高斯柔阴影 + 细描边”体现轻拟物层级。
- 关键反馈（成功/支付）加入短时高光 sweep 动效，建立仪式感。

## 4. 图形语言
- 圆角体系：卡片 10px、输入 8px、按钮 12px。
- 插画形态：以圆角几何、弧线轨迹、轻量图标构建，不使用写实照片。
- 图标风格：线性 + 小面积填色，线宽 1.75px，圆角端点。

## 5. 字体与排版
- 字体：`Microsoft YaHei`, `PingFang SC`, `Source Han Sans CN`, sans-serif。
- 标题强调：H1/H2 采用高对比，正文确保长文可读。
- 数字强调：价格、匹配度、评分统一使用半粗体 + 更高对比色。

## 6. 页面氛围层级
- L1（品牌层）：Hero 背景光感 + 插画。
- L2（内容层）：卡片、表单、列表。
- L3（交互层）：按钮、悬浮助手、Toast、弹层。
- L4（反馈层）：加载骨架、空态、成功/错误状态区。
