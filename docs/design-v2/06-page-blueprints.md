# 悟空哆哆 H5 Design V2 · 页面蓝图

> 说明：所有关键页面均需覆盖 `default/loading/empty/error/success` 五态。

## 1. 首页 `HomePage`
- 布局：Hero → 场景入口 → 智能助手 → 信任背书 → 动态信息。
- 主 CTA：`开始智能匹配`
- 次 CTA：`浏览四大场景`
- 动效：Hero 渐变浮动 + 卡片错峰入场。

## 2. 找项目 `FindProjectPage`
- 结构：目标说明 → 表单 → 智能补全 → 报告摘要 → 候选列表。
- 行动：`生成项目报告`、`匹配开发者`、`发起支付`。
- 错误态：预算区间异常、时间缺失、匹配失败。

## 3. 找家教 `FindTutorPage`
- 结构：需求输入 → 智能补全 → 候选筛选 → 联系方式支付。
- 行动：`匹配家教`、`获取联系方式`。
- 成功态：展示联系方式获取成功卡片。

## 4. 找陪练 `FindSparringPage`
- 结构：训练目标输入 → 智能补全 → 候选列表 → 预约支付。
- 行动：`匹配陪练`、`立即预约`。
- 成功态：显示训练计划摘要卡片。

## 5. 发现兼职 `DiscoverJobsPage`
- 结构：技能输入 → 智能补全 → 人才画像 → 职位推荐 → 申请。
- 行动：`生成人才画像`、`推荐岗位`、`申请兼职`。
- 空态：暂无岗位时显示“优化画像建议”。

## 6. 候选详情 `CandidateDetailPage`
- 结构：基础信息、标签、经验、亮点、行动按钮。
- 行动：`返回列表`、`立即选择`。

## 7. 支付中心 `PaymentPage`
- 结构：订单摘要、支付面板、状态时间线、历史记录。
- 行动：`确认支付`、`取消支付`、`重试支付`。
- 成功态：支付成功动画 + 下一步提示。

## 8. 消息中心 `MessagesPage`
- 结构：消息分类筛选 + 消息列表 + 批量已读操作。
- 行动：`刷新`、`全部已读`。

## 9. 个人中心 `ProfilePage`
- 结构：用户身份卡、返利统计、快捷入口、成长记录。
- 行动：`去认证`、`发起提现`。

## 10. 实名认证 `AuthPage`
- 结构：角色选择 + 身份信息 + 提交反馈。
- 行动：`提交认证`。
- 错误态：字段缺失、格式无效。

## 11. 投诉反馈 `ComplaintsPage`
- 结构：投诉场景 + 对象 + 原因 + 证据 + 提交结果。
- 行动：`提交投诉`。
- 成功态：投诉编号与后续处理说明。
