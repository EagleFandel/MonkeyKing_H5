# 悟空哆哆 H5 Design V2 · 差距矩阵

## 1. 视觉系统差距
| 规范要求 | 当前实现 | 差距等级 |
|---|---|---|
| 多层色阶 token | 仅少量基础色变量 | P1 |
| Hero 品牌叙事视觉 | 首页为普通卡片堆叠 | P0 |
| 阴影/描边分层规则 | 阴影单一，层级不明显 | P1 |

## 2. 组件契约差距
| 组件 | 规范要求 | 当前实现 | 差距等级 |
|---|---|---|---|
| `CandidateCard` | variant/density/motionPreset | 仅固定样式 | P0 |
| `DodoAssistantPanel` | mode/state/contextHint | 仅基础输入+结果 | P0 |
| `PaymentSheet` | intent/statusTone/successAnimation | 仅渠道选择+确认 | P0 |

## 3. 交互与状态差距
| 要求 | 当前实现 | 差距等级 |
|---|---|---|
| 页面转场策略 | 无路由转场体系 | P1 |
| 关键动作反馈节奏 | 基本依赖 toast | P1 |
| 5 态覆盖（所有关键页） | 多数页面缺空态/错误态组件化表达 | P0 |

## 4. IA 与内容差距
| 要求 | 当前实现 | 差距等级 |
|---|---|---|

| 场景页五段式统一 | 结构相近但未系统化 | P1 |
| 行动层级（主次 CTA） | 部分页面 CTA 不突出 | P1 |

## 5. 可访问性差距
| 要求 | 当前实现 | 差距等级 |
|---|---|---|
