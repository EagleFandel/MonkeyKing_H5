# 悟空哆哆 H5 Design V2 · 缺陷清单

## P0（必须立即修复）
1. 首页缺失品牌 Hero 叙事结构，首屏模板感强。
2. 三大关键组件未具备设计契约扩展能力：`CandidateCard` / `DodoAssistantPanel` / `PaymentSheet`。
3. 关键页面 5 态覆盖不完整，空态/错误态表达缺失。
4. 支付成功反馈缺少仪式性与下一步引导。
5. 多页面中文文案存在编码异常（显示乱码），严重影响体验。

## P1（本轮应修复）
1. 全局 token 未形成完整色阶、动效与层级体系。
2. 缺路由级 pageTone/motionProfile，跨页风格跳变。
3. 表单输入反馈弱（focus/error/success 视觉状态不足）。
4. 消息中心、个人中心信息密度与层级表现偏弱。
5. 缺少统一状态组件，页面重复实现导致不一致。

## P2（可延后优化）
1. 局部插画细节与动效精修。
2. 更丰富的数据可视化（趋势图、完成率图）。
3. A/B 方案对比与体验实验。
